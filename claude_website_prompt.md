# model Prompt -- Performance Website Layout Generator

Copy and paste this prompt into model (Anthropic) to generate a modern
responsive performance-band portfolio website.

------------------------------------------------------------------------

## Prompt for model

Build a modern, responsive portfolio-style performance website layout
using clean semantic HTML and Tailwind CSS (or React + Tailwind if
preferred). The website represents a live performance band / event music
collective.

The website must include the following sections with smooth scrolling
navigation and professional animations.

------------------------------------------------------------------------

## Navigation Menu

Home\
About Us\
Gigs\
Portfolio\
Gallery\
Contact

Navbar behavior:

-   Sticky navigation
-   Transparent on hero section
-   Turns solid on scroll
-   Mobile hamburger menu

------------------------------------------------------------------------

## Hero Section (Full Screen Background Video)

Create a cinematic landing section with:

-   Fullscreen autoplay muted background video
-   Dark overlay gradient
-   Centered headline text
-   Supporting subtitle
-   CTA button

Headline:

Crafting Stories Through Sound & Live Experiences

Subtitle:

Live Performances • Wedding Events • Concert Shows • Studio Production

CTA Button:

Explore Our Work

Button must smoothly scroll to the main content section.

Mobile fallback:

Use background image if video cannot autoplay.

------------------------------------------------------------------------

## Main Intro Section

Create a centered introduction section describing services.

Example content:

We create unforgettable live musical experiences for weddings, concerts,
corporate events and curated stage productions.

Feature cards:

Live Performances\
Wedding Events\
Corporate Shows\
Studio Production

Cards must include hover animation effects.

------------------------------------------------------------------------

## About Us Section

Create a split layout section:

Band / performance team image placeholder

Right:

stage experiences across weddings, concerts, corporate shows and curated

Optional stats row:

Years of Experience\

------------------------------------------------------------------------
